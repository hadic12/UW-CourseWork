Estimation Answers

1. O(g(n)) = O(n^3) 
2. O(g(t)) = O(t^5) 
3. O(g(x)) = O(x^2)
4. O(g(w)) = O(w)

Polynomials Answers

1. f(x) = 1
2. f(x) = n^2 + 1
3. f(x) = log(n) + 1

Demonstration Answers

1. f(x) = 1, g(x) = 1, c = 1, k = 0 or 1.
2. f(x) = n^2 + 1, g(x) = n, c = 1, k = 0 or 1.
3. f(x) = log(n) + 1, g(x) = log(n), c = 1, k = 0 or 1.
