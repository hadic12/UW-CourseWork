Q. 3)

Q. Step Over:
A: Go to the next line in the block.

Q. Step Into:
A: Go to the next line or into a user defined method.

Q. Step Out:
A: Completes and moves to to the point that called the method

Q. Continue:
A: Runs the program from where programmer has set debug point up to the end of the program or to the point where programmer has set break point
